# Sanvaya - Student & Organizer Event Portal

A full-stack web application for managing student events, enabling organizers to create and manage events while allowing users to register and track participation through a structured system.

---

## Overview

The application provides a centralized platform to handle event workflows including creation, updates, and user registrations. It follows a REST-based architecture with a clear separation between frontend, backend, and database layers.

---

## Features

- Create, update, and delete events  
- User registration for events  
- Participation tracking  
- Responsive user interface  
- Structured data storage using MongoDB  

---

## Setup

### Clone the repository

```bash
git clone https://github.com/Ragi-Varshini-Reddy/ACM-Event-Portal.git
cd ACM-Event-Portal
```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
